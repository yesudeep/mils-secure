  • If you are a corporate user...

    Legacy systems that work only using Internet Explorer 6 still keep the Web from growing and evolving further. Major companies like Google have stopped supporting Internet Explorer 6 or earlier.

    Many companies have policies that do not allow people to install new software on their computers. We think this is very short-sighted of their administrators. You can use a portable version of Firefox which does not require any installation and can be run directly without administrative privileges.

    Many times administrators refuse to install new software into your computer simply because it's more work for them! They'll give you all kinds of "technical" excuses why you shouldn't install another browser. Don't let that happen to you. Please try a different browser before forming a judgement.

    Just have your administrator install a newer browser alongside Internet Explorer 6 and use Internet Explorer 6 only for those legacy Web applications. It's that simple.
